# Bluewild-money
Bluewild 경제 시스템 플러그인

기능: 상점 거래 같은 거 까지만 됨

명령어: /coin

명령어: /shop
관리자 권한 필요

TMI: 함정이 있다. /shop 명령어를 통해 소환한 npc는 reload/restart 시 사라지며
상점 매뉴를 열 때 소환된 npc는 가상 npc이기 때문에 매뉴가 안 열린다. 열려면 명령어로

§e§lspecial shop
§6§l정육점
§9§l엔§1§l더§9§l드§1§l래§9§l곤
§c§l지옥 상점
§7§l광부

이와 같은 이름으로 생명체를 소환 후 클릭하여 열어야 한다.
